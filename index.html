Binary Counter

Electronics Project

This project is a pulse counter and indicator, built with a 555 timer and a CD4040 decade counter, using simple and readily available electronic components. Its main goal is to count a series of pulses generated by the timer and visually display the accumulated count in binary format through light-emitting diodes (LEDs). The project serves as a practical application of fundamental electronic principles, including timing circuits, digital counting, and binary representation.

  • Role:Embedded System Engineer
  • Technologies: 555 Timer, CD4040 IC, Custom PCB
